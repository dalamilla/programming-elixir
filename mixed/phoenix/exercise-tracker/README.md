# Exercise Tracker

Exercise Tracker microservice implemented on Phoenix.

## Instructions

To start this app:

- Installing Dependencies

```
mix deps.get
```

- create and migrate the database:

```
mix ecto.setup
```

- Running Phoenix Application:

```
mix phx.server
```

- Run Phoenix Test Application:

```
mix test
```
