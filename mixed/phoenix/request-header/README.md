# Request Header Parser

Request Header Parser microservice implemented on Phoenix.

## Instructions

To start this app:

- Installing Dependencies

```
mix deps.get
```

- Running Phoenix Application:

```
mix phx.server
```

- Run Phoenix Test Application:

```
mix test
```
